## Running

### Installation

```shell
$ npm install
```

### Development mode

```shell
$ npm start
```

### Production build

```shell
$ npm build
```

## Database

* generate migrations:

```shell
$ npm run migration:generate
```

* run migrations:

```shell
$ npm run migration:run
```
